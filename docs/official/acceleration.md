# Minecraft 连接 IP

本页面提供 LPS 服务器的 Minecraft 连接 IP 信息。

---

## 如何使用？

### 基本步骤

1. **复制加速 IP 地址**：从下方列表中复制适合您的加速 IP
2. **添加到服务器列表**：
   - 打开 Minecraft 客户端
   - 进入"多人游戏"（Multiplayer）
   - 点击"添加服务器"（Add Server）
   - 在"服务器地址"中粘贴连接 IP
   - 输入服务器名称（如：LPS - 加速通道）
   - 点击"完成"（Done）
3. **连接服务器**：双击服务器列表中的新条目即可连接

### 注意事项

- ⚠️ 请确保使用与服务器版本匹配的客户端版本
- ⚠️ 如果加速 IP 无法连接，请尝试使用主服务器 IP
- ⚠️ 部分加速 IP 可能是减速器，看清延迟

### 根据地理位置选择

- **境内**：晚上选择加速节点，早上移动可以不用

### 方法一：使用游戏客户端测试

1. 在服务器列表中，选择加速 IP 条目
2. 查看右侧显示的延迟（ms）
3. 延迟越低表示连接越快

## 常见问题

### Q: 加速 IP 和官方 IP 有什么区别？

A: 

- **官方 IP**：服务器的官方地址，稳定性高延迟以及丢包较高
- **加速 IP**：经过优化的备用地址，可能提供更低的延迟。

### Q: 我应该使用加速 IP 还是官方 IP？

A: 

- 如果官方 IP 连接正常且延迟可接受，可以继续使用官方 IP
- 如果官方 IP 延迟较高或不稳定，建议尝试加速 IP
- 可以将多个 IP 都添加到服务器列表，根据需要切换

### Q: 加速 IP 无法连接怎么办？

A: 

1. 检查客户端版本是否匹配
2. 尝试其他加速 IP 节点
3. 使用官方 IP 连接
4. 检查网络连接

### Q: 加速 IP 会定期更换吗？

A: 

- 加速 IP 地址通常保持稳定，不会频繁更换
- 如有变更，会在公告中通知
- 建议定期查看 [Wiki](wiki.md) 获取最新信息

### Q: 使用加速 IP 需要额外配置吗？

A: 

- 通常不需要额外配置，直接使用 IP 地址和端口即可
- 某些情况下可能需要配置代理，请参考相关教程

### Q: 加速 IP 安全吗？

A: 

- 加速 IP 由服务器虽然个人提供，安全性没有保障
- 但是就一转发盗不了你号，担心 MOD 比这重要多了
- 但是还是不要使用来源不明的 IP 地址，以防账号安全问题

## 使用技巧

### 1. 多个 IP 同时添加

建议将主 IP 和适合的加速 IP 都添加到服务器列表，这样可以：

- 在主 IP 出现问题时快速切换到加速 IP
- 根据当前网络情况选择最佳连接
- 对比不同节点的延迟和稳定性

### 2. 定期检查更新

- 定期查看本页面，获取最新的加速 IP 信息
- 关注服务器公告，了解 IP 变更情况
- 加入 [QQ群](../community/qq.md)获取实时更新

### 3. 网络优化建议

除了使用加速 IP，还可以：

- 使用有线网络连接（比 Wi-Fi 更稳定）
- 关闭占用带宽的其他应用
- 配置客户端优化参数（参考 [Wiki](wiki.md)）
- 本地开启bbr（没啥大提升）
