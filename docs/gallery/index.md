## 图片展示示例

以下是 Material for MkDocs 支持的图片展示方式：

### 方式一：标准 Markdown 图片

```markdown
![图片描述](assets/img/gallery/example.jpg)
```

### 方式二：带链接的图片

```markdown
[![图片描述](assets/img/gallery/example.jpg)](assets/img/gallery/example.jpg)
```

### 方式三：使用卡片布局

```markdown
<div class="grid cards" markdown>

-   ![建筑1](assets/img/gallery/buildings/1.jpg)

    **标题**：建筑名称

-   ![建筑2](assets/img/gallery/buildings/2.jpg)

    **标题**：建筑名称

</div>
```

### 方式四：图片网格（使用 Material 特性）

Material for MkDocs 支持通过自定义样式实现图片网格布局，可在 `mkdocs.yml` 中配置相关 CSS。
