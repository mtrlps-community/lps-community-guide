# 如何进入服务器？

## 准备

一台正常工作并且互联网连接正常的电脑 （储存空间>5g） 
一个正版mc账号 （离线不用抱着试试态度，网易版也不行）  
智慧或者善于思考的大脑  

## 安装教程

### 安装启动器
首先下载pcl2/hmcl启动器 官方启动器无法自动装整合包所以不在教程内容中  
**pcl2启动器**：`https://afdian.com/p/0164034c016c11ebafcb52540025c377`  
**hmcl启动器**：`https://hmcl.huangyuhui.net/`  
这两个哪个顺眼下哪个就行  

### 配置启动器
#### pcl2  
选择正版 登录你的mc账号  
点击版本选择 选择合适的文件夹存放游戏  

#### hmcl  
选择账户 登录你的mc账号  
点击实例列表选择合适文件夹存放游戏  


### 安装游戏  
点击下载 选择mc版本 1.20.4 安装farbic 版本选第一个就行 安装fabric api 版本也是选第一个  
等待安装  
**下载mod包**：`维护中`  
下载好的直接拖入启动器 启动器自动安装

### 启动  
启动游戏选择多人游戏尽情享用吧  
如没有服务器请见 [连接ip](official/acceleration.md)
